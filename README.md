# Guru Jambheshwar Mandir Parta - Website

A clean, responsive, and modern website for Guru Jambheshwar Mandir Parta temple built with HTML, CSS, and JavaScript.

## Features

✨ **Full-Screen Image Slider** - Auto-sliding with manual controls
📱 **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
🎨 **Modern Design** - Clean UI with reddish-yellow theme colors
🎥 **YouTube Integration** - Embedded videos and channel links
🗺️ **Google Maps** - Location mapping for easy navigation
⌨️ **Keyboard Navigation** - Arrow keys to navigate slider
👆 **Touch Swipe** - Swipe gestures for mobile slider navigation
🍔 **Mobile Menu** - Hamburger menu for smaller screens
🔄 **Smooth Transitions** - Elegant animations and transitions
♿ **Accessible** - Respects prefers-reduced-motion preference
